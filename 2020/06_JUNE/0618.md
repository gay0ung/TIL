<p><strong>for…in / for …of</strong></p>
<blockquote>
<p>for in 반복문 : 객체의 모든 열거 가능한 속성에 대해 반복<br>
for of 반복문 : [Symbol.iterator] 속성을 가지는 컬렉션 전용</p>
</blockquote>
<pre><code>let myString = "test";

// logs 0, 1, 2, 3
for (let i in myString) {
     console.log(i);
 }

// logs t, e, s, t
for (let i of myString) {
    console.log(i); 
}
</code></pre>
