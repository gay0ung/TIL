### ◼ switch / case
```javascript
function  doOperation()  {
	const  intValueA  =  parseInt(firstValue,  10);
	const  intValueB  =  parseInt(secondValue,  10);
	
	switch  (currentOperation)  {
	case  "+":
		return  intValueA  +  intValueB;
	case  "-":
		return  intValueA  -  intValueB;
	case  "/":
		return  intValueA  /  intValueB;
	case  "*":
		return  intValueA  *  intValueB;
	default:
		return;
	}
}
```
- if 문같이 사용할수 있다.
- break 가 있다면 해당 스위치 블록을 빠져나온다.
