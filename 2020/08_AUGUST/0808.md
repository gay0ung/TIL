## localStorage
##### key값 읽기
```js
var cat = localStorage.getItem('myCat');
```
##### 선택한  key값 삭제
```js
localStorage.removeItem('myCat');
```
##### 저장된 key값 모두 삭제
```js
// Clear all items
localStorage.clear();
```
