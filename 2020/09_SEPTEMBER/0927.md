## reverse()
```js
 Array.reverse()
```
### 반환값
: 순서가 반전된 배열
### 설명
reverse 메서드는 호출한 배열을 반전하고 원본 배열을 변형하며 그 참조를 반환합니다.
#### ex
```js
const a = [1, 2, 3];
console.log(a); // [1, 2, 3]

a.reverse();
console.log(a); // [3, 2, 1] 
```
